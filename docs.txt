# 🧩 Project: **Component Playground (UI Builder Tool)**

---

## 🧠 1. Product Idea (what you’ll say to recruiters)

> “A visual UI builder that allows developers to create, customize, and export reusable components with live preview and clean code output.”

👉 This instantly shows:

* System thinking
* Reusability mindset
* Real-world frontend skills

---

## 🎯 2. Core Features (MVP → Advanced)

### ✅ MVP (must-have)

* Component selector (Button, Card, Input)
* Live preview panel
* Controls panel (color, padding, radius, font)
* Code export (JSX + Tailwind)
* Responsive preview (mobile/tablet/desktop toggle)

---

### 🚀 Advanced (this is where you shine)

* Copy-to-clipboard + “Copied!” animation
* Theme switch (light/dark)
* Save component config (localStorage)
* Share via URL (state in query params 🔥)
* Variant system (primary, secondary, outline)
* Accessibility checker (contrast warning)

---

## 🧱 3. Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **State:** Zustand (clean + simple)
* **Icons:** Lucide
* **Code formatting:** Prettier (for export)
* **Optional:** shadcn/ui (for base components)

---

---

## 🧠 4. State Design (Zustand)

```ts
{
  selectedComponent: "button" | "card" | "input",

  styles: {
    backgroundColor: string,
    textColor: string,
    padding: string,
    borderRadius: string,
    fontSize: string
  },

  variant: "primary" | "secondary" | "outline",

  device: "mobile" | "tablet" | "desktop",

  updateStyle: (key, value) => void,
  setComponent: (component) => void
}
```

## ⚙️ 7. Code Generator Logic

```ts
export function generateButtonCode(styles) {
  return `
<button className="
  ${styles.backgroundColor}
  ${styles.textColor}
  ${styles.padding}
  ${styles.borderRadius}
">
  Click me
</button>
`;
}
```
